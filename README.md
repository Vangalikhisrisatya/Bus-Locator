# 🚌 Bus Locator

A Flask-based web application developed to help students locate their assigned college buses and view their real-time or latest available location. The system reduces waiting time and improves communication between students and bus operators.

---

## 📌 Project Overview

The Bus Locator system allows students to check the location of their assigned bus through a simple web interface. Administrators can manage bus details, routes, and student assignments, making transportation management more efficient.

---

## ✨ Features

- Student login
- View assigned bus details
- View bus location
- Bus route information
- Admin dashboard
- Manage buses
- Manage student records
- Responsive web interface
- Secure database storage

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Vangalikhisrisatya/Bus-Locator.git
```

### 2. Navigate to the project folder

```bash
cd Bus-Locator
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the application

```bash
python app.py
```

### 7. Open in your browser

```
http://127.0.0.1:5000
```
